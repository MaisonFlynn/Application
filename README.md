## Installation

1. **Clone Repository**
    ```sh
    git clone https://github.com/username/repository.git
    cd repository
    ```

2. **Install Dependencies**
    ```sh
    npm install
    ```

3. **Create `.env` File IN Root w/ Variable(s)**
    ```plaintext
    CONNECTION=MongoDB Connection String
    USER=Email
    PASS=Email's Password
    KEY=JWT Secret Key
    PORT=Preferred Port, Default 3000
    ```
